# Учёт занятий (Teacher Journal)

Оффлайн-приложение для преподавателя: учёт **занятий** (единица учёта — одно
проведённое занятие, НЕ часы), посещаемости, оценок, ведомостей и отчётов.
Полностью офлайн, без API/ключей/сети. База — SQLite.

## Возможности
- Группы, студенты (в т.ч. массовый ввод списком), предметы (план в занятиях).
- Неповторяемость данных: дубли групп/студентов/предметов/занятий блокируются
  ограничениями UNIQUE в БД.
- Отметка занятия: вопрос «Занятие проведено?»; отметка присутствия нажатием на
  имя; оценки (наличие оценки = присутствовал); неотмеченные = отсутствовали.
- Ведомость (средний балл) + диаграммы (на Kivy Canvas, без matplotlib).
- Отчёт учёта занятий: план / проведено / осталось / %.
- Экспорт отчётов в CSV/PDF и расписания в `.ics` (импорт в Google/Яндекс календарь).
- Локальные напоминания о незаполненных занятиях (plyer).

## Запуск на ПК
```bash
pip install kivy plyer pillow
python main.py
```
> `pillow` нужен для рендеринга текста в Kivy; `plyer` — для уведомлений
> (на десктопе может требовать доп. бэкенд, отсутствие не критично).
