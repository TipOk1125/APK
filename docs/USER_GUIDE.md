# Руководство пользователя

## F1. Группы, студенты, предметы
1. «Группы и студенты» → введите название группы → «Создать группу».
2. Выберите группу из списка → добавьте студента по ФИО.
3. «Предметы и расписание» → выберите группу → введите название предмета и
   **план в занятиях** → «Создать».

## F2. Массовое добавление студентов
«Массовое добавление студентов» → выберите группу → вставьте список (каждая
строка = один студент) → «Сохранить список». Появится итог:
«Добавлено N, пропущено дублей M».

## F3. Неповторяемость / выбор из списка
Везде студент/предмет/группа сначала **выбираются** из списка. Создание —
отдельная кнопка. Повторный ввод того же имени блокируется.

## F4. Напоминания и отметка занятия
При запуске приложение проверяет незаполненные прошедшие занятия и показывает
их количество (и локальное уведомление). Экран «Отметить занятие»:
выберите группу и предмет, дату/время (пусто = сейчас) и ответьте
«Занятие было проведено?»:
- **Нет** — занятие помечается как не проведённое и **не** идёт в учёт.
- **Да** — откроется список студентов для отметки.

## F5. Отметка присутствия
- **Нажатие на имя** переключает присутствие (зелёный = присутствовал).
- Кнопка **«Оценка»** — выбор оценки 2–5; наличие оценки автоматически = присутствовал.
- Все **неотмеченные** студенты по умолчанию = **отсутствовали**.
- «Сохранить» фиксирует посещаемость.

## F6. Ведомость и отчёты
«Отчёты и ведомость» → выберите группу/предмет:
- **Ведомость** — пофамильно средний балл, число оценок, посещения без оценки,
  плюс столбчатая диаграмма средних баллов.
- **Учёт занятий** — план / проведено / осталось / % (в занятиях).
- **Экспорт CSV / PDF** — файл сохраняется в каталог `TeacherJournal`
  (на Android — во внешнем хранилище; на ПК — в `exports/`).

## F7. Экспорт расписания в календарь (.ics)
На главном экране — «Экспорт расписания в календарь (.ics)». Файл `schedule.ics`
сохраняется в каталог `TeacherJournal`.

### Импорт в Google Calendar
1. Откройте calendar.google.com → шестерёнка → «Настройки».
2. Слева «Импорт и экспорт» → «Импорт».
3. Выберите файл `schedule.ics`, укажите календарь → «Импортировать».

### Импорт в Яндекс.Календарь
1. Откройте calendar.yandex.ru.
2. Слева внизу «Импортировать события из файла .ics» (или Настройки → Импорт).
3. Выберите `schedule.ics` → подтвердите импорт.

> На телефоне можно просто открыть файл `.ics` — система предложит добавить
> события в календарь по умолчанию.
